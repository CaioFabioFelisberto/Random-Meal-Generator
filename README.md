# 🥘 Gerador de Receitas Aleatórias (TheMealDB API)

Um aplicativo Python simples e resiliente que busca receitas aleatórias de uma API externa e as exibe de forma organizada.

## 🚀 Tecnologias
* **Python 3.x**
* **Requests**: Para consumo da API REST.
* **HTML/CSS**: Interface gerada com auxílio de IA.

## 🧠 O que eu aprendi
* **Tratamento de Exceções**: Uso de `try/except` para lidar com falhas de conexão e erros de API (HTTP 404/500).
* **Performance com Strings**: Uso do método `.join()` para concatenar ingredientes de forma eficiente, respeitando a imutabilidade das strings em Python.
* **Alinhamento de Dados**: Lógica para unir ingredientes e medidas vindos de chaves separadas no JSON.

## 🛠️ Como rodar
1. Clone o repositório.
2. Instale as dependências: `pip install requests`
3. Execute: `python main.py`
